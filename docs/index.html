🆕 What's New / Update

新たに、CVE-2026-31431がCISAのKnown Exploited Vulnerability（KEV）カタログに追加され、実用的なPoC exploitの存在も確認されています。脆弱性はLinux kernelのalgif_aead/AF_ALG周辺の不具合で、非特権ユーザーが読み取り可能ファイルのpage cacheを4バイト書き換え、setuidバイナリを悪用してroot権限を得られる可能性があります。CVSSは7.8（High）で、2017年以降のカーネルに広く影響し、container breakoutや横展開にもつながり得ます。Microsoft Defenderは検知とハンティング手法、緩和策を提示し、今後追加情報に応じて更新予定です。

💡 Key Takeaway

日本の金融・公共・製造業でLinuxやKubernetesを使う環境では、単一ノードの侵害がホスト全体や隣接コンテナへ波及するリスクがあります。SSH侵入、CIジョブ悪用、コンテナ侵害を起点にroot昇格される可能性があるため、該当カーネルの迅速なパッチ適用、露出サービスの最小化、Defender XDRによる検知確認を優先すべきです。
