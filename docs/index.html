Microsoft Foundry Agent Service、Observability と Network Isolation が GA

📌 概要

Microsoft Foundry Agent Service は2025年10月に Preview として公開された Microsoft のマネージドエージェント基盤で、エンタープライズ向けの AI エージェント開発・運用を統合的に支える製品。今回 GA に伴い、エンタープライズ採用に必要なObservability および Private Networking 機能群が一斉に解禁された。

🆕 What's New / Update

今回の発表で Observability と Private Networking の一般提供（GA）が宣言された。Hub-Project モデルに完全対応し、VNet 統合および Network Security Perimeter 経由でのプライベート接続が可能に。エージェントの実行トレース、トークン使用量、レイテンシをネイティブに可視化できる新ダッシュボードも搭載。日本リージョン（東京・大阪）でも同時 GA となり、データ常駐性要件を満たした本番運用が可能となった。APIM AI Gateway との統合も強化され、レート制限・監査ログがマルチプロバイダで統一管理できる。

💡 Key Takeaway

金融・公共セクターのデータ常駐要件を満たせるため、本番採用が現実的に。特に既存の Foundry Classic 利用顧客は2026年内の新ポータル移行検討が必要。日本国内のメガバンクや官公庁案件で PoC を進めている顧客は、この GA を契機に本格展開フェーズに入れる。Observability の標準実装により、エージェント運用の可観測性確保コストが大幅に下がる点も重要。
