💡 Key Takeaway

microsoft.com/security — Microsoft Security Blog
📌 概要

Microsoft Defender XDR が、Linux カーネルの crypto-subsystem に存在する高深刻度のローカル権限昇格脆弱性 CVE-2026-31431（Copy Fail）を調査中と公表した記事です。Red Hat、SUSE、Ubuntu、AWS Linux など複数ディストリビューションに影響し、クラウド Linux や Kubernetes 環境での被害拡大が懸念されています。

🆕 What's New / Update

新たに、CVE-2026-31431 が CISA の Known Exploited Vulnerability（KEV）カタログに追加され、実働する PoC の存在とともにパッチ競争が始まっています。脆弱性は Linux kernel の algif_aead / AF_ALG における処理不備で、非特権ユーザーが読み取り可能なファイルの page cache を破損し、/usr/bin/su などを介して root 権限を取得できるとされています。CVSS は 7.8（High）で、Microsoft Defender は予備的な試験的活動を観測し、検知・ハンティング・緩和策の案内を提供しています。

💡 Key Takeaway

この脆弱性はリモート単体ではなくローカル侵入後に危険度が跳ね上がるため、SSH、CI/CD、コンテナ侵害を起点とした横展開対策が重要です。金融・公共・製造の Kubernetes／共有 Linux 基盤では、パッチ適用、影響カーネルの棚卸し、Defender XDR による検知強化を急ぐべきです。コンテナ越しにホストへ波及し得る点が特に実務上のリスクです。
